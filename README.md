# 🍱 今天吃什麼？- 隨機美食決定器 (What To Eat)

![Language](https://shields.io)
![Status](https://shields.io)

**「今天吃什麼」** 是一個專為解決選餐困難而設計的實用工具。透過趣味的大富翁與拉霸機模式，幫你快速從口袋名單中選出下一餐！

🔗 **[👉 立即點擊體驗網頁](https://github.io)**

---

## 🌟 核心功能

*   **🕹️ 雙遊戲模式**：內建「正餐大富翁」與「夜市拉霸機」，讓決定美食變有趣。
*   **💾 自動記憶功能**：你的自定義美食清單會自動儲存在瀏覽器中，下次開啟無需重新輸入。
*   **📱 手機優化介面**：專為行動裝置設計，載入極快且支援單手流暢操作。
*   **🎯 直觀結果顯示**：結果直接顯示在大富翁格子下方，方便查看與記錄。

---

## 🚀 如何使用

1. **選擇模式**：在網頁上方切換正餐、點心或夜市模式。
2. **調整清單**：在輸入框輸入餐廳名稱並點擊「新增」，或按 **✕** 刪除不想要的選項。
3. **開始決定**：
   - **大富翁模式**：點擊中間的「選一個」按鈕。
   - **拉霸機模式**：點擊右側的「拉霸手把」。
4. **結果揭曉**：畫面會自動顯示今天的命定美食！
